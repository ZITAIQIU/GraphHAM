# GraphHAM
This repository contains the source code and dataset for our paper **"An Efficient Automatic Meta-Path Selection for Social Event Detection via Hyperbolic Space"**.

# Requirements
* Python>=3.7
* torch>=1.4.0
* scipy>=1.2.1
* networkx>=2.4
* scikit-learn>=0.20.3

# Usage
## To run GraphHAM
```python run.py```
